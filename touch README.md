# MultiSource CloudStream Plugin

A CloudStream plugin that provides access to multiple streaming sites:

- **MusicHQ** (https://musichq.cc) - Movies and TV shows
- **MovieMaze** (https://moviemaze.cc) - Movies and TV shows
- **KickassAnime** (https://kickass-anime.ru) - Anime content
- **MirrorSite** (https://a.111477.xyz) - Mirror content provider

## Features

- Search across multiple sites
- High-quality video streaming
- Episode support for TV series and anime
- Recommendation system
- Multiple video source extraction

## Installation

1. Download the latest release from the [Releases](https://github.com/cyberm-del/moviexyz/releases) section
2. In CloudStream 3, go to Settings > Extensions > Install from file
3. Select the downloaded `.cs3` file
4. Restart the app to see the new providers
